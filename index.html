Project Name: WorldSurf

Project Description

This site will serve as an information portal to the public providing the highest visibility of content on various surf destinations around the world. Users will be able to explore detailed information of each destination. Each destination will include "members only" content, providing WorldSurf.com members with bookmark capability, web cam access to the destination, as well as will allow users to create a free account. With an account, members will have access to the "members only" content for each destination, as well as access WorldSurf.com. The access given to members will aid in usability testing.
